## Blockchain go

Basic prototype implementation of Blockchain 1.0.

Written in golang 1.12.


### Run Project  

* Build project using go
    ```sh
    go build
    ./blockchain-go
    ```

* Or Build project using make 
    ```sh
    make run
    ```
